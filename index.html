Welcome

This project is the a lightweight pill manager which lets users add pills to a DOM container and then be able to extract the value of the collection later.

Examples:

var pills = new PillCollection($('#pill-collection'));
pills.addPill('pill1 value');
pills.addPill('pill2 value', 'Pill text');
pills.reset();
pills.reset([['pill3'], ['pill4', 'Fourth pill text']]);

History

The excellent JS library Chosen greatly advanced UX for large lists in select inputs. The use of pills on multi-selects is beautiful and intuitive. I wanted to be able to use pills in other widgets so this library was built.
